# Original-LinearAlgebra-textbook
## まず...
![ダウンロード](https://github.com/user-attachments/assets/dc67abe5-4a57-440e-8c60-d8d2eca963e1)




線形代数学の学習はStrangの本が最初でした。(https://github.com/Hinatori8/Trefethen-Bau_notes でも記述)
大学の履修科目で「秀」を取るために始めましたが、この本は線形代数学の面白く感じたきっかけとなりました。そこで、2年前期では履修科目の「線形代数学」の試験範囲を扱った初学者向けのテキストを執筆して後輩に公開しておりました。
このテキストは、

- 線形代数学の面白さを共有したい
- 自分の知識や理解不足している箇所の整理
- 自己満足

を目的としていました。

## なぜ作ったの？
添付されたpdfの参考文献の**前期線形代数学**と**後期線形数学**の2冊作成しておりました。これは公開しません。個人情報が含まれているからです。2年後期に入って、「ここまで書いてきたのであるならば、卒業するまでに自分の名前が入った本を1冊出したい」との思いに駆られ、完全版を作成するに至りました。

## 本書の構成

1. 「前期線形代数学」と「後期線形数学」の連結(公開しないと述べたが、公開可能な場所を残し、9割そのまま。ただし、再編に際して、説明文章を書き直している。)
2. 外積や内積といった、周辺知識の追加掲載
3.  試験範囲ではないものの、知っておくと得をしそうな内容を拡充。いずれも**発展**として分けて章立てしている。

特に**発展**に関しては、試験対策の枠組みを超えて、「テキスト」としての役割を持ってもらうために、大幅に拡張しました。しかし、専門性が高すぎて使わないような内容は載せないようにしました。
